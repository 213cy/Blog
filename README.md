$$\Large\left{\begin{eqnarray} \frac{dx}{dt}&=&Px-Q\\\frac{dy}{dt}&=&Py\end{eqnarray}\right.$$


x:星体发出的光线到地球观测点的距离
y:发出光线的星体到地球观测点的距离
P:宇宙膨胀速率
Q:光速

上式解得:


$$\Large\left{\begin{eqnarray} x&=&C_1e^{Pt}+\frac{Q}{P}\\y&=&C_2e^{Pt}\end{eqnarray}\right.$$

t=0 时,x=y=147亿光年
t=147亿年,x=0,y=460亿光年

模型完了.
